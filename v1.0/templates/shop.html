Как пользоваться

Смена роли и пользователя

  1. Выбери роль в правом верхнем углу.
  2. Для роли `user` обязательно укажи `client_id` или нажми один из demo-профилей.
  3. Нажми "Применить", чтобы открыть стенд в новом контексте доступа.
  4. Для `manager` доступны товары и аналитика, для `admin` доступно управление клиентами.

Что проверить

Основные сценарии

  • Покупатель Фильтрация каталога, корзина, создание заказа
  • Менеджер Создание товара, смена статуса заказа, аналитические отчеты
  • Администратор Создание, редактирование и удаление клиентов
  • API Swagger на `/docs` и живая MongoDB под контейнерами Docker
