Обязательные API-методы

  • GET /products, POST /products, DELETE /products/{product_id}
  • GET /branches, POST /branches, DELETE /branches/{branch_id}
  • GET /employees, POST /employees, DELETE /employees/{employee_id}
  • GET /health возвращает объединённый статус MongoDB и PostgreSQL

Ручная проверка

  • Откройте страницы управления через верхнее меню.
  • Создавайте и удаляйте документы в MongoDB на странице «Товары».
  • Создавайте и удаляйте записи в PostgreSQL на страницах «Филиалы» и «Сотрудники».
  • Используйте /docs для Swagger и прямых API-вызовов.

Подсказки по запуску

Запустите стенд командой docker compose up -d --build. Масштабируйте API через docker compose up -d --scale api=2. Для удаления контейнеров и томов обеих БД используйте docker compose down -v.
